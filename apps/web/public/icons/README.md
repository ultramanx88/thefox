# PWA Icons

Place your PWA icons in this directory with the following sizes:

## Required Icons
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## Shortcut Icons
- shortcut-markets.png (96x96)
- shortcut-orders.png (96x96)
- shortcut-profile.png (96x96)

## Apple Touch Icons
- apple-touch-icon.png (180x180)

## Favicon
- favicon.ico (32x32)

## Design Guidelines
- Use the theFOX brand colors (#ff6b35 theme color)
- Ensure icons are maskable (safe area within 80% of the icon)
- Use simple, recognizable designs
- Test on various backgrounds (light/dark)
