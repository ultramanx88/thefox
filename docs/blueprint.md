# **App Name**: TaladMan

## Core Features:

- Marketplace Browsing: Browse marketplace listings and vendor profiles.
- Vendor Registration & Product Listing: Register as a vendor and list products for sale, including location-based service offerings.
- Shopper Registration: Register as a personal shopper. Location services will connect shoppers and customers.
- Real-Time Order Tracking: Integrated system that enables real-time tracking for both vendors and customers.
- Rating & Review System: Provide an integrated rating and review system, enhancing marketplace transparency.
- AI Category Suggestion Tool: Suggest product categories using AI, based on an image provided by the user. This feature acts as a tool to help the user determine which product category is most appropriate.

## Style Guidelines:

- Primary color: Vivid green (#22C55E) for marketplace freshness.
- Background color: Light green (#E6F4EA), subtly tinted to provide a clean backdrop that does not compete with the primary color.
- Accent color: Burnt orange (#D97706), an analogous color with slightly reduced brightness and saturation to maintain visual harmony, for CTAs and highlights.
- Body and headline font: 'PT Sans' (sans-serif), suitable for both headlines and body text, offering a modern look with a touch of warmth, ideal for a marketplace platform.
- Clean and consistent icons representing product categories and actions.
- Mobile-first, responsive design for seamless use on various devices.
- Subtle animations for loading states and user interactions to improve engagement.
